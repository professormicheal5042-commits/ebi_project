[ ] TASK 10 (2:00 – 2:30)  Python OCR API Setup with FastAPI + Vercel (FREE)
    ─────────────────────────────────────────────────────────────
    STEP 1 — Create the FastAPI project folder:
        - Inside the project root, create a new folder: `api/`
        - Create `api/index.py` with the following code:

            from fastapi import FastAPI, File, UploadFile
            from fastapi.middleware.cors import CORSMiddleware
            import pytesseract
            from PIL import Image
            import io, re

            app = FastAPI()
            app.add_middleware(CORSMiddleware, allow_origins=["*"],
                               allow_methods=["*"], allow_headers=["*"])

            @app.post("/api/extract-expiry")
            async def extract_expiry(file: UploadFile = File(...)):
                contents = await file.read()
                image = Image.open(io.BytesIO(contents))
                text = pytesseract.image_to_string(image)
                # Parse date (DD/MM/YYYY or MM/YYYY)
                date_match = re.search(r'\b(\d{2}[\/\-]\d{2}[\/\-]\d{4}|\d{2}[\/\-]\d{4})\b', text)
                expiry = date_match.group(0) if date_match else "Not found"
                # Try to grab product name (first non-empty line)
                lines = [l.strip() for l in text.splitlines() if l.strip()]
                name = lines[0] if lines else "Unknown"
                return {"name": name, "expiry": expiry, "raw_text": text}

=========================================================================
  PYTHON AI / OCR INTEGRATION (ARCHITECTURE EXPLANATION)
=========================================================================
  While the UI demo currently simulates OCR with a loading spinner, the
  actual AI intelligence is designed to be handled by Python:
  
  1. Image Capture: User clicks the image picker in `add_product.html` and uploads a photo of the product label.
  2. Data Transfer: The frontend sends this image to a Python backend (e.g., Flask/FastAPI) or Supabase Storage.
  3. OCR Processing: Python runs Optical Character Recognition (e.g., Tesseract, EasyOCR, or Google Vision) to read the raw text.
  4. AI Parsing: Python uses an LLM or regex to parse the raw text, extracting the exact "Product Name" and "Expiry Date".
  5. Response: Python returns structured JSON `{ "name": "Milk", "expiry_date": "2026-10-15" }` to the frontend.
  6. Auto-fill & Save: The UI auto-fills the form fields. The user verifies it, clicks "Save", and the data is saved permanently to the Supabase Database.
